# Node Web App Project
Still in development
https://c12018fnpj.herokuapp.com/
StMedi Spa business website with Client Management System.

## Getting Started

Run the following commands in your terminal, after cloning the repo:

```
> cd client
> npm install
> cd ..
> npm install
> npm run dev  
```

## Built With

* Express.js
* React.js
* Redux
* MongoDB
* reduxForm
* Heroku
