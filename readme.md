# Final Project
Still in development

StMedi Spa business website with Client Management System.

## Getting Started

Run the following commands in your terminal, after cloning the repo:

```
> cd FinalProject
> cd client
> npm install
> cd ..
> npm install
> npm run dev  
```

## Built With

* Express.js
* React.js
* Redux
* mongoDB
* reduxForm
